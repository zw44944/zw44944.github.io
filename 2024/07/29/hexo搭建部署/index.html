hexo搭建部署

Git安装

Node.js安装

1.hexo安装

npm install hexo-cli -g

2.本地建站

2.1 初始化

新建一个存放博客的目录，进入终端执行

hexo init myblog cd myblog npm install

vscode 打开myblog，目录显示如下

_config.yml存放的是博客配置信息，source/_posts是存放文章的地方。

3.更换主题

4.部署到Github Pages

登录 GitHub，新建一个 Repository，Repository name一定要是你的用户名.github.io

来到仓库主页，复制如下链接

打开博客目录下的_config.yml，拉到最后，填写deploy模块：repository 填写刚才复制的链接

一定要安装 Git 插件：

npm install hexo-deployer-git --save

现在可以将博客部署上去了，执行

hexo c && hexo g && hexo d # hexo d 表示执行部署

执行成功可以通过 你的用户名.github.io 来访问博客了。

5.数据迁移

更换电脑进行数据迁移时，需要将这些博客原文备份至新电脑，因此原文的保存十分重要。
